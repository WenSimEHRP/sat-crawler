# SAT Questions Crawler

Simple yet _worksful_ question crawler.

## Usage

### UNIX/Linux/MacOS

Install dependencies via `pip3 install -r requirements.txt`.

```sh
./crawl.py
```

### Windows

Install dependencies via `pip3 install -r requirements.txt`.

```sh
python3 ./crawl.py
```

### Using Nix/NixOS

Use `nix-shell -p` to enter the development environment.

```sh
./crawl.py
```
